# API de Livros

## Descrição
API REST completa com CRUD de livros.

## Tecnologias
- Node.js
- Express
- SQLite

## Endpoints

### GET /livros
Lista todos os livros

### GET /livros/:id
Busca por ID

### POST /livros
Cria um livro

### PUT /livros/:id
Atualiza totalmente

### PATCH /livros/:id
Atualiza parcialmente

### DELETE /livros/:id
Remove um livro
